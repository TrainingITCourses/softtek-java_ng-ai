# Linting Code Skill

When asked for linting code, follow these steps:

## Steps to Lint Code

### 1. Identify the codebase:
  - Determine which files or directories need to be linted.
  - Use git history to find recently modified files if no specific path is provided.
### 2. Check the Linter
  - Check if there is a linter configured for the projects 
  - If not, suggest installing and setting up a popular linter.
    - Read and use the [Adding Linters](./2-3-5-adding-linters.md) instructions.
### 3. Run the linter:
  - Use the appropriate linter for each tech stack. 
  - Run the linter on the identified files or directories.
