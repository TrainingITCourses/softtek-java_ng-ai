# Vibe health check

Seguir un briefing (español o inglés)

## Conversación para crear mensaje de bienvenida y salud del sistema.

> Genera un mensaje de saludo que incluya el nombre de la aplicación y sirva como comprobación del estado del sistema. El mensaje debe indicar explícitamente que el servidor está operativo e incluir un timestamp actual.

## Conversación para crear la funcionalidad de gestión de cohetes

> Genera un API restful y una aplicación web para gestionar cohetes, incluyendo las siguientes operaciones: crear un nuevo cohete, obtener la lista de cohetes, actualizar la información de un cohete específico y eliminar un cohete.
